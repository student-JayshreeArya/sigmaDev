1. Flex Direction - It defines in which direction the flex elements would be displayed. It takes values like row, column or “reverse” too.

2. Flex Wrap - By using this property we can make our elements responsive for different screen sizes.

3. Justify Content - This property is used to set the position of content or rather align content along the main axis.

4. Align Items - Just like the justify-content property, align-items define the alignment of the flex container but along the cross-axis. (This property used when having single line)

5. Align Content - This property is very similar to align item but here rather than the flex items, the content present in the item is selected for the property. (This property used when having multiple line)
